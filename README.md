# 📚 Flashcard App — Тренажёр иностранных слов

Консольное приложение на Go для изучения иностранных слов с помощью карточек (flashcards). Приложение позволяет создавать карточки, проводить тренировки, отслеживать статистику и сохраняет все данные в JSON файл

## ✨ Возможности

- **Управление карточками**: добавление, просмотр, удаление карточек
- **Тренировка**: случайный порядок карточек, отслеживание правильных/неправильных ответов
- **Статистика**: процент успешных ответов, карточки, требующие повторения
- **Персистентность**: автоматическое сохранение всех данных в JSON файл
